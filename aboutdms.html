Donor Management System

Donor Management System is a Desktop based end to end Web Application made on Visual Studio Code and pgadmin 4 (PostgreSQL).

In summary, what I built is:

  • Using this application an organization can manage and maximise the relationships with the Donors.
  • The various HR features included are - Dashboard page for Visual Understanding of activities of Donors, Campaign Management, Donor Management, Donor Activities, Report Genration etc.
