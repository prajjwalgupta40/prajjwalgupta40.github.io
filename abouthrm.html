Human Resource Management System

Human Resource Management System is a Desktop based end to end Web Application made on Visual Studio Code and pgadmin 4 (PostgreSQL).

In summary, what I built is:

  • Using this application an organization can manage and integrate the various HR process
  • The various HR features included are - Employee Compensation, Attendance, Employee Management, Payroll, Leave Management, Appraisals etc.
  • This application also provides a pannel for employees on which they can login and apply for Leave Request which will be later approved/rejected by the admin.
